Helpful Tools and External Resources

On this page, you'll find links to tools and resources to assist you in your HTML, CSS, and JavaScript journey. These tools and resources were chosen to help students both learn introductory front-end web design concepts and provide "I am making a website right now and need help" tools. Another factor in selecting these tools and resources were their accessibility and functionality. The core functionalities of these tools and resources are provided without charge or required account creation.

It is important to note that many of the links on this page will be sorted into the category that best fits them. There are inevitable overlaps between HTML, CSS, and JavaScript, so don't take our categorizations as law.

Note: TagTutor is not sponsored by nor affiliated with any listed tools or resources. While the team at TagTutor curated these tools and resources, use the provided tools and resources at your own risk.

MDN Web Docs
Formally known as the Mozilla Developer Network and the Mozilla Development center, MDN docs is an open-source documentation repository for web development technologies. The website provides resources to learn HTML, CSS, and JavaScript, as well as references for accessibility, web APIs, web extensions, and other types of web technologies.
Use MDN Web Docs as reference while you're building your website, or learn from its educational modules.
W3Schools
W3Schools is a popular beginner resource for learning HTML, CSS, JavaScript, and more. Despite its name, W3Schools is not officially associated with the World Wide Web Consortium (W3C). The website provides step by step tutorials, reference documentation, and interactive quizzes for beginner learners of various front-end and back-end web development technologies.
Use W3Schools as reference while you're building your website, or learn from its educational modules, tutorials, and quizzes.
W3docs
Similar to W3Schools and MDN Web Docs, W3docs is a web development learning resources for beginners to intermediates. Despite its name, W3docs is also not officially associated with the World Wide Web Consortium (W3C).
Use W3docs to learn through its interactive quizzes and exercises, example code snippets, and various web development tools like color mixers and CSS makers.
Codrops
Codrops is a website that for web design enthusiasts and developers looking for inspiration and innovation in interactive web design. The website features a rich collection of developer spotlights, interactive demonstrations, highlights, and articles about web development.
Use Codrops to find inspiration for your projects from CSS effects to JavaScript libraries to bring your project to the next level.
Practical Typography by Matthew Butterick
Practical Typography is a web-based book on typography written by Matthew Butterick, a writer, typographer, programmer, and lawyer. The book provides key rules and quick lessons on typographic concepts for people who want practical, straightforward (and opinionated) rules for good typography.
Use Practical Typography to learn the key rules of typography and elevate the appearance of your website's text.
NN/g
The Nielsen Norman Group, stylized as NN/g, is a consulting firm that specializes in user experience (UX) design. Its website hosts a library of free articles and videos on the field of UX and how to best design web pages with user experience in mind.
Use NN/g's library of free articles to learn the best practices for designing web pages that are both appealing and pleasant to use.
GeeksforGeeks
GeeksforGeeks is a comprehensive education resource that covers fields from computer science to programming to web technologies. While GeeksforGeeks provides introductory resources for front-end beginners like its HTML Cheat sheet, the site provides information that goes beyond the basics.
Use GeeksforGeeks to learn all about how to bring your front-end and/or back-end skills to a new level.
Prettier
Prettier is an opinionated code format that formats code into a consistent style. Prettier is opinionated, meaning it uses its own format, not allowing users full customization over code formatting.
Use Prettier to clean up your code's appearance and make it look "pretty." (For anyone working on a group project, this may be a good tool for you!)
Coolors
Coolors is a color palette generator with features like contrast ratio checking, community palette collections, and palette saving. Coolors also has tools like its gradient maker, gradient collection, and image converter.
Use Coolors to quickly generate a color palette for whatever design you need.
Can I Use
Can I Use is a website that provides up-to-date information on browser support for front-end web technologies. Can I Use provides percentages of users whose browsers support a certain feature, as well as list the supported versions of the supported browsers.
Use Can I Use to ensure that your website works for whoever views it.

W3C's Markup Validation Service
W3C's Markup Validation Service allows users to validate their HTML by providing a URI to the CSS document, by uploading a file, or by directly inputting the markup into a web form. Validation ensures that the markup is compliant with the official guidelines set out by W3C.
Use the W3C's official markup validator to validate your HTML files.
Shape Divider
Shape Divider is a browser-based shape generator that allows users to create custom SVG dividers to add to websites. Users can customize the divider's shape, color, height, and more, using Shape Divider's interactive generator.
Use Shape Divider to create custom SVG decorations for your websites.

W3C's CSS Validation Service
W3C's CSS Validation Service allows users to validate their CSS by providing a URI to the CSS document, by uploading a file, or by directly inputting the CSS into a web form. Validation ensures that the CSS is compliant with the official guidelines set out by W3C.
Use the W3C's official CSS validator to validate your CSS.
CSS Grid Generator
CSS Grid Generator is an interactive, browser-based tool that allows users to create custom grids. After creating a custom grid, users can examine and copy the generated HTML and CSS code.
Use CSS Grid Generator to interactively learn how CSS grid works, or use just it to save the headache of setting up a complex grid.
Flexbox Froggy
Flexbox Froggy is a game all about moving a frogs onto lilypads, teaching players how to use CSS flexbox. The game requires players to utilize different flexbox container and flexbox item properties to affect the location of frogs.
Use Flexbox Froggy to learn the basics of CSS Flexbox and how items flow in a flex container.
Grid Garden
Grid Garden is like Flexbox Froggy. Grid Garden is a game all about moving a water onto crops, teaching players how to use CSS flexbox. The game requires players to utilize different grid container and grid item properties to affect the location of the water square.
Use Grid Garden to learn the basics of CSS grid and how to move items in a grid container.
btns
btns is an open-source button designs with hover effects. Click on the button design to see the HTML and CSS involved in creating and styling the button.
Use btns to elevate your button designs on whatever projects you make.
CSS-Tricks
CSS-Tricks is a website dedicated to CSS. From guides and news to inspiration and tutorials, CSS-Tricks is a treasure trove of CSS-related information.
Use CSS-Tricks to learn the fundamentals of CSS or take your CSS to the next level.
flexbox.help
Use flexbox.help to test CSS flexbox rules using the website's interactive flexbox model. Users can add or remove flexbox items, change the flex direction, and more.
Use flexbox.help to interactively learn how CSS flexbox container properties affect the alignment and flow of child items.
