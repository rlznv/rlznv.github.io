Welcome to the HTML home page

Here, you'll find our HTML Main Guide and our HTML Tricky Concepts pages. These pages were created to be supplemental roundups of the most important HTML concepts you'll need for ENGL 3368.

The Main Guide covers topics like basic HTML document structure, common tags, and inline vs. block elements. Our Tricky Concepts page discusses beginner student struggles and provides solutions and suggestions for dealing with these struggles.
