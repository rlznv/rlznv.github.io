The Anatomy of an HTML Element

It is important to close an element because it allows for the website to know the start and end of each element. A tag consists of an element name placed within brackets. Each element uses a opening and closing tag to contain the element; the difference between the tags is the closing tag uses a slash before the element name is enclosed within the brackets. 

<elementname>Content here.</elementname>

Basic Document Structure

The text file cannot become an HTML file without the basic document structure. Without this basic document structure, the website will not read any characters or symbols correctly, causing it to render on the page incorrectly. 

  <!DOCTYPE html> *The first line in the HTML document shows the website the document type and version of HTML used, allowing for proper website rendering.
  <html>
  <head>
  <meta charset= “UTF-8”> *This tag tells the browser to use the UTF-8 character encoding when displaying the content. UTF-8 represents most of the characters and symbols and allows for browsers not to misinterpret characters.
  <title>Title goes here.</title> *The title of the website is written here.
  </head>
  <body>  *The body tag is a block-level element.
    Page content goes here. *Headers, paragraphs, images, etc., are placed into the body of the document.
  </body> 
  </html>

Default Name Files

The default file is called the index file. The default file name for an HTML file is index.html, but the file name can vary depending on how the server is configured. Titling the website's main page as index.html provides an easy starting point and allows for easy movement between the index page and subpages within the website.

HTML TAGS

Heading Elements

Headings can be defined as <h1> through <h6>. They can be used to structure information on a website, allowing for an optimal flow for the reader to follow. Heading elements are also block elements.

<h1> H1 </h1> 
<h2> H2 </h2>
<h3> H3 </h3>
<h4> H4 </h4>
<h5> H5 </h5>
<h6> H6 </h6>

Thematic Breaks

Thematic breaks can create a horizontal line across the page. This can be used to seperate content or to show a thematic break. You can use CSS to customize the appearance of the line break, such as the thickness and color. 

<hr>

Lists

All list tags are block-level elements by default. There are 3 main lists that can be used in a website.

  • Unordered List
    • An unordered list typically consists of a bulleted list of items.
  • Ordered List
    • An ordered list can be a numerical or alphabetical list.
  • Description List, Description Term and Description Value
    • Description list tag displays a list of terms along with a corresponding description. The tag can act as a container for the entire list.
    • Description term is used within the description kist to represent a term that is being described.
    • Description value is also used within the description list to represent the description of the term. 
<ul> Unordered Lists </ul>
<ol> Ordered Lists </ol>
<dl> Description Lists </dl>
<dt> Description Term </dt>
<dd> Description Value </dd>

Long Quotations

Paragraph

A paragraph tag helps structure content allowing seperation of text blocks. 
<p> Paragraph </p>

Blockquote

Blockquote is used for defining a section of a text that is in a long quotation from a diffrent source. The cite attribute can be also used. It specifies the URL of the source for the quote that was used. Blockquote usually stands alone and is separated from other surrounding text. 

<blockquote> Blockquote </blockquote>

Preformatted Text

Any text displayed using the preformatted tag will be on a fixed-width font. It allows for rendering of space, line breaks and tabs. It is useful when showing code snippets, art or when having precise formatting is important.

<pre> Preformatted Text </pre>

Figures

Figures are inline elements used for contained content, such as an image or diagram. They can also provide a description of the content.

<figure> Figure </figure>

Figure Caption

Figure caption describes the contents of the figure element and allows for an easy access to the description.

<figcaption> Figure Caption </figcaption>

Inline vs Block Elements

Inline Element

Inline elements allows the application of CSS to be able to style the element. Inline elements are typically used for small HTML files or quick edits. This allows for quick styling without affecting other elements.

<span> Inline Element </span>
  <em> Emphasis </em>
  <strong> Strong </strong>
  <small> Small </small>
  <br> Break </br>
  <var> Variable </var>
  <i> Italic </i>
  <b> Bold </b>
  <img> Image </img>

Block Element

Block elements occupies the full width of the container, allowing for vertical stacking of elements. Block elements can be stacked side by side horizontally using CSS.

<div> Block Element </div>
  <aside> Aside </aside>
  <nav> Navigation </nav>
  <header> Header </header>
  <footer> Footer </footer>
  <blockquote> Blockquote </blockquote>
  <pre> Preformatted Text </pre>

Giving Structure to your HTML File

Main Content

The website's main content should be placed within the main tag element. This shows the user and web browsers that this is where the important information is located on the website. It allows for the main part of the content to be in one section, it also shows the important of the information when it is in the main tag.

<main> Main Content </main>

Header

A header is an introductory container where the navigation and website name can be contained in. It allows for users to find information quick and easy.

<header> Header </header>

Footer

A footer can display information at the bottom of the page such as copyright, contact information and navigation elements. It allows for users to find information quick and easy.

<footer> Footer </footer>

Sections

Sections are used for thematic grouping of content. By default the section tag is a block-level element. It can also help divide sections of the page.

Sections of similar content within the page are grouped together this allows for easy accessibility to users and browsers. 

<section> Section </section>

Articles

Articles can be used for small-contained sections such as in blogs, news articles or where content is independent from one another.

<article> Article </article>

Sidebars

Aside

The aside element can be used to create side bars on a website. It holds information that is not directly related to the main content of the page but can be set aside independently from it.

Side bars can be used for side menus and information boxes that are separate from the main content of the website.

<aside> Aside </aside>

Navigation

Navigation provides links with the current document or for outside documents as well. It allows for easy accessibility because it contains the navigation links for the website. It can improve user usability along with enhancing the websites' structure.

<nav> Navigation </nav>

Images

Images can be used to imbed an image on the website by linking the image to the website. Src makes a specified path to the image while alt makes a specified alternate text for the image incase the image cannot be displayed.

<img src= “URL” alt= “Description”>
