Creating a new document in TextEdit (macOS)

Step 1

Open TextEdit application on your macOS. It can be found using Finder then searching through the Applications Folder.

Step 2

            
Once you Open TextEdit, click the New Document button. If you see a text formatting menu and a tab ruler you are in rich-text mode. If not you are in plain-text mode.

Step 3

            
Close the document and head towards the top of your macOS system. Next to the Apple symbol, the TextEdit tab should be next to it. Hover over the TextEdit and click on Settings.

Step 4

            
On the New Document tab click on plain text. Under Options uncheck all of the automatic formatting options. On the Open and Save tab, select Display HTML files as HTML Code 5. Deselect "Add '.txt.' extension to plain text files."

Step 5

            
The rest of the defaults should be good as they are. When you are done, click on the button of the settings box.

Step 6

            
Now you can click on the File tab next to TextEdit and click New to create a New Document. The original settings are reverted to the new changes.

Note: You can convert the document back to rich-text mode when you are not using TextEdit for HTML.

Why do we change the settings for TextEdit?

Changing the settings for TextEdit allows for proper rendering of the code on the website.
