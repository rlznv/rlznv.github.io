JavaScript Main Guide

Introduction

JavaScript allows for different interactions on the website such as visual effects, animations, and transitions. This can help websites be more user friendly and bring different design elements that elevate the aesthetical look of the website.

Attaching JavaScript to HTML Documents

Embedded Script

Embedded Script allows for a piece of code written in JavaScript to be integrated within the website.

 <script> Embedded Script </script>

External Script

External script can be used when the same code is being used in multiple web pages.

 <script src> "my_script.js" </script>

Script Placement in HTML Code

To be able to properly put JavaScript in with your HTML code you will need to use the script tag.

  <!DOCTYPE html>
  <html>
  <head>
  <meta charset= “UTF-8”>
  <title>Title goes here.</title>
  </head>
  <body>
   ...Content of the page...
  <script scr="script.js"> </script>
  </body> 
  </html>

Script Anatomy

To properly write or recognize JavaScript code you will need to know the anatomy of a script. The script is made up of different parts that are used to create the code. Here are common terms to become familar with:

Statements

A statement is a command line that tells the browser to preform a specific task or action. These statements are what make up a script. Statements typically end with a semicolon but a line break is also able to end the statement. For better readability and organization, it is recommended to end the statement with a semicolon.

alert("Thank you.");

Comments

In a script, comments are used to help a reader understand the code. This is a good practice to include in your code especially if you are working with a team or the code will be edited in the future by someone else. Comments are not read on the browser and are not part of the code. There are two types of methods to use comments:

Single Line Comments

To create a single line comment, place two slash characters at the start of the line.

// This is a single line comment.

Multi-Line Comments

To create a multi-line comment, place a slash and an asterisk at the start of the line and an asterisk and a slash at the end of the line.

/* This is a multi-line comment.
The script will ignore this when it preforms its task */

Variables

Variables are similar to containers that hold information. This information can be a number, a string, or a boolean. Variables are used to store data values that can be used later in the code. To declare a variable, you will need to use the var keyword followed by the name of the variable. The name of the variable can be anything but it is recommended to use a name that describes what the variable is storing. Note that Javascript and variables are case-sensitive. Here are important rules to follow when giving a variable a name:

  • The name must begin with either a letter or an underscore.
  • The name can include letters, digits, and underscores, in any order.
  • The name cannot include spaces hence the use of underscores.
  • The name cannot use any special characters such as:
    • !
    • .
    • ,
    • /
    • \
    • +
    • *
    • =

Data Types

Undefined

This is when you give a variable a name but not a value. This will cause the variable to have a value of undefined. This will be useful for when you are troubleshooting your code, when a variable shows as undefined it usually is not declared correctly or there is a typo.

var foo;  
alert(foo);

Null

Null is similar to undefined, although this one defines the variable but does not having any meaningful value.

var foo= null;
alert(foo);

Integers

Variables can be assigned integers. You can use the variable name as if it was its value. For example, foo can be considered as the number 5. You can write (foo + foo).

var foo= 5;
alert(foo);
alert(foo + foo);

Strings

Strings creates a line of text in which the characters are in a single or double set of quotes defines it into a string.

var foo= "five";
alert(foo);

Booleans

With the boolean variable, you can create true or false values.

var foo= true;

Arrays

Arrays are a group of values that can all be assigned to a single variable. When listing the values they need to refer to a set of numbers based on how they appear on the list. The first one is assigned the number 0, then 1, and so on.

var foo= [1, 2, 3, 4, 5];
alert(foo[0]); 
alert(foo[1]); 
alert(foo[2]);
