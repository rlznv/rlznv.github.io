JavaScript Main Guide

What is JavaScript used for?

Using JavaScript allows for different interactions on the website such as visual effects, animations, and transitions. This helps websites be more user friendly and brings different design elements that elevate the aesthetical look of the website.

How to attach JavaScript to HTML code?

Embedded Script

Embedded Script allows for a piece of code written in JavaScript to be integrated within the website.

 <script> Embedded Script </script>

External Script

External script can be used when the same code is being used in multiple web pages.

 <script src> "my_script.js" </script>
