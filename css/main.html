CSS: Cascading Style Sheets

Cascading Style Sheets (CSS) is a stylesheet language used to apply style and presentation to a document. It allows web developers to create visually appealing and responsive designs, enhancing user experience and accessibility.

Attaching Style to a Document

Attaching a style to a document benefits user experience as it allows for a consistent and visually appealing interface. Style attachment can be added through various methods: inline, embedded, and external.

External CSS

External styles are linked to the document using a <link> tag, allowing for separation of content and presentation. External style sheets are preferred for larger projects as it promotes reusability and maintainability of styles across multiple pages.

<style>@import url(external.css);</style>

Embedded CSS

Internal styles are defined within a <style> tag in the document's head section.

<style>@...</style>

Inline CSS

Inline styles are applied directly to HTML elements using the "style" attribute.

<h1 style= "color: red; margin-top: 2em">@...</h1>

CSS Terminology

Selectors

Selectors are patterns used to select the elements you want to style. They can be simple or complex, allowing for precise targeting of elements.

Declarations

Declarations are the individual style rules that define how the selected elements should be styled. Each declaration consists of a property and a value.

Properties

Properties are the specific attributes of an element that can be styled, such as color, font-size, and margin.

Values

Values are the specific settings assigned to properties, determining how the element will be styled.

Color

Color is a fundamental aspect of web design, allowing for the creation of visually appealing and engaging interfaces. CSS provides various methods for defining colors, including named colors, hexadecimal values, RGB, RGBa, HSL, and HSLA.

Color Names

Color names are predefined color values that can be used in CSS. They are easiest to remember and most convenient to use, but they are limited in number.

Hexadecimal

Hexadecimal color values are a six-digit combination of numbers and letters that represent colors. They are widely used in web design due to their precision and versatility. Below is the structure of the hexadecimal RGB value:

#RRGGBB

Where RR, GG, and BB are two-digit hexadecimal values representing the red, green, and blue values of the colors. Each value can range from 00 to FF. For example, the color white is represented as #FFFFFF, while black is represented as #000000.

RGB

RGB color values are defined using the RGB function, which takes three parameters representing the red, green, and blue components of the color. The values can be specified as integers ranging from 0 to 255 or as percentages.

HSL

HSL color values are defined using the HSL function, which takes three parameters: hue, saturation, and lightness. The hue is represented as an angle in degrees (0-360), while saturation and lightness are represented as percentages.

Fonts

Fonts are essential for typography and readability in web design. Consider accessibility practices when choosing a font.

Font Family

The font-family property is used to specify the typeface of the text. It can accept multiple values, allowing for fallback options in case the preferred font is not available.

Font Size

The font-size property is used to set the size of the text. It can accept various units, including pixels, ems, rems, percentages, and viewport units.

Pesudo-classes

Pesudo-classes are special keywords added to selectors that specify a special state of the selected elements. They are used to style elements based on their state, such as hover or focus.

Anatomy of the Pseudo-class Selector

The selector is signified by a colon (:) character then followed by the the name of the pseudo-class. Here are a few commonly used selectors:

  • a:link { }
    • applies a style to unclicked (unvisited) links
  • a:visited { }
    • applies a style to links that have already been clicked
  • a:focus { }
    • applies when the element is selected and ready for input
  • a:hover { }
    • applies when the mouse pointer is over the element
  • a:active { }
    • apply styles to an element while it is in the process of being activated
