CSS Flexbox

Introduction

The CSS Flexible Box Layout Module, or Flexbox, is a one dimensional layout system for rows and columns. It provides responsive design advantages such as making all neighboring items the same height, creating simple horizontal and vertical centering, and changing the order in which the items are displayed.

  • Flex Item Properties

Flexbox Creation

Step One: Setting up a Flexbox Container

In the body of the HTML text file, create a div tag with a class titled corresponding to the name of your boxes. This will be the container for your flexbox. Inside the div, create a number of divs with different class names for each box. Each of these will be the boxes inside the flexbox. Markup Example:

<body>
  <div id="container">
    <div class="box box 1">1</div>
    <div class="box box 2">2</div>
    <div class="box box 3">3</div>
    <div class="box box 4">4</div>
    <div class="box box 5">5</div>
  </div>
</body>

In the CSS style sheet, identify the container using the div id name and set the display to flex. By default, the div tags display as block-level elements; therefore, the box items are set to display vertically in the container. Setting the display to flex modifies the flex items to display as inline elements.
Style Example: 

#container{ display: flex;}

Step Two: Specifying Flow Direction and Wrap

Now that the Flexbox is created, you can adjust the flow direction and wrap of the container and items.

We recommend trying out flexbox.help's interactive flexbox generator to test out flexbox's different behaviors.

Flow Direction

To control the flow of the flex items, the flex-direction property must be modified. The default value for the flex-direction property is row, which arranges the flex items from left to right. This property is be placed in the container id within the stylesheet.

  • flex-direction: row;
    • The row value modifies the direction to arrange horizontally from left to right.
  • flex-direction: row-reverse;
    • The row-reverse value modifies the direction to arrange horizontally from right to left.
  • flex-direction: column;
    • The column value modifies the direction to arrange vertically from top to bottom
  • flex-direction: column-reverse;
    • The column-reverse value modifies the direction to arrange vertically from bottom to top.
Flex Wrap

The flex-wrap property allows the flex items to break off or wrap into a new line. The default value for the property is nonwrap, which disables wrapping. The flex-wrap is placed in the container id within the stylesheet.

  • flex-wrap: nonwrap;
    • The nonwrap value disables wrapping.
  • flex-wrap: wrap;
    • The wrap value allows the flex items to wrap into a new line.
  • flex-wrap: wrap-reverse;
    • The wrap-reverse value allows the flex items to wrap into a new line in reverse order.
Flex Flow

A property that merges flex-direction and flex-wrap into one is flex-flow. The flex-flow property can be used for either or both the direction or wrap of the flex items. The default value for the property is row nonwrap, which orders the flex items from left to right and disables wrapping. The property is be placed in the container id within the stylesheet. There are 16 possible values for the property, here are the most commonly used:

  • flex-flow: row nonwrap;
    • The row nonwrap value orders the flex items horizontally from left to right and disables wrapping.
  • flex-flow: row wrap;
    • The row wrap value orders the flex items horizontally from left to right and allows wrapping.
  • flex-flow: column nonwrap;
    • The column nonwrap value orders the flex items vertically from top to bottom and disables wrapping.
  • flex-flow: column wrap;
    • The column wrap value orders the flex items vertically from top to bottom and allows wrapping.

Step Three: Aligning the Flex Items

Aligning on the Main Axis

To align the flex items on the main axis, use the justify-content property which alters how space within the container moves around or between the flex items. The default value for the property is flex-start which positions the items at the start of the main axis. This property can be used no matter what value is set to the flex-direction of the container. The alignment will apply to the container whether it is arranged horizontally or vertically. The property will be placed in the container id within the stylesheet.

  • justify-content: flex-start;
    • The flex-start value positions the items at the start of the main axis.
  • justify-content: flex-end;
    • The flex-end value positions the items at the end of the main axis.
  • justify-content: center;
    • The center value positions the items in the center of the main axis.
  • justify-content: space-between;
    • The space-between value positions the items with space between them.
  • justify-content: space-around;
    • The space-around value positions the items with space around them.
Aligning on the Cross Axis

To align the flex items on the cross axis, use the align-items property, which adjust the position and stretching of the items in the container. The default value for the property is stretch, which stretches the flex items from top to bottom depending on the container's set height value.

  • align-items: stretch;
    • The stretch value stretches the flex items from top to bottom depending on the container's set height value.
  • align-items: flex-start;
    • The flex-start value positions the flex items at the start of the cross axis.
  • align-items: flex-end;
    • The flex-end value positions the flex items at the end of the cross axis.
  • align-items: center;
    • The center value positions the flex items in the center of the cross axis.
  • align-items: baseline;
    • The baseline value positions the flex items at the baseline of the cross axis.
Multiple Line Alignment

To align multiple lines within a flex container, use the align-content property, which modifies the alignment for multiple flex lines. In order to use the property, the container must have a set height. The property can only be applied when the flex-wrap property has a set value to wrap or wrap-reverse and there are multiple flex lines. The default value is stretch, which evenly distributes the flex items in the container. This property will be placed within the stylesheet in the Flexbox's container id.

  • align-content: stretch;
    • The stretch value evenly distributes the flex items in the container.
  • align-content: flex-start;
    • The flex-start value positions the flex items at the start of the cross axis.
  • align-content: flex-end;
    • The flex-end value positions the flex items at the end of the cross axis.
  • align-content: center;
    • The center value positions the flex items in the center of the cross axis.
  • align-content: space-between;
    • The space-between value positions the flex items with space between them.
  • align-content: space-around;
    • The space-around value positions the flex items with space around them.

Flex Item Properties

Flex properties are applied to flex items instead of the container. The values of the flex properties include the properties: flex-grow, flex-shrink, and flex-basis. When creating the markup for the flex property, be sure to order the values like so:

flex: flex-grow flex-shink flex-basis;

For example, if the flex-grow value is set to 1, the flex-shrink value is set to 1, and the flex-basis value is set to auto, the markup would be written as:

  .box1 {flex: 1 1 auto;}

Item Expansion

The flex-grow property allows for flex items to expand to fill the available space in the container. The default value for the property is 0, which disables expansion. The property is placed under a new id for the singular flex item in the stylesheet. Ensure that the id name matches the id given in the HTML markup. The flex-grow value is the first value written in the flex property.

Item Shrinkage

The flex-shrink property allows for flex items to shrink to fit the container. The default value for the property is 1, which allows the flex items to shrink. The property is placed under a new id for the singular flex item in the stylesheet. Ensure that the id name matches the id given in the HTML markup. The flex-shrink value is the second value written in the flex property.

Item Basis

The flex-basis property allows for flex items to set a base size. The default value for the property is auto, which sets the base size to the width of the item. The property is placed under a new id for the singular flex item in the stylesheet. Ensure that the id name matches the id given in the HTML markup. The flex-basis value is the third value written in the flex property.

Item Order

The order property has the ability to reorder flex items. The style markup is placed under a new id for the singular flex item in the stylesheet. Ensure that the id name matches the id given in the HTML markup. The default value for the property is 0, which orders the flex items in the order they are written in the HTML markup. The flex items are reordered by assigning a positive or negative integer value to the property. The lower the number, the higher up in the order it will be placed.

For example, if you want to move box 5 to be first in line, you would assign it a value of -1 and all other boxes a value of 0 or higher.

  .box4 {order: -1}

Item Alignment

To align individual flex items on the cross axis, use the align-self property, which applies the set value to the specified items. The default value for the property is stretch. The property is placed under a new class for the singular flex item in the stylesheet. Ensure that the class name matches the id given in the HTML markup.

.box4 {align-self: flex-end;}
