CSS Grid

Introduction

The CSS Grid is a two-dimensional layout system for creating rows and columns. CSS Grid allows for complex layouts although it is not as flexible as Flexbox. Grid is best used for larger scale layouts.

Table of Contents

  • Grid Terminology
  • Grid Creation
  • Grid Behavior

Grid Terminology

To fully understand why and how we use the CSS Grid Layout, you must become familiar with the commonly used terms.

  • Grid Container: The parent element that contains the grid items.
  • Grid Item: The child elements inside the grid container.
  • Grid Track: The space between two adjacent grid lines. It can be a row or a column.
  • Grid Cell: The space between four adjacent grid lines. It is the smallest unit of the grid.
  • Grid Line: The lines that form the structure of the grid. They can be horizontal or vertical.
  • Grid Area: The space enclosed by grid layout; area inside the container. It can contain one or more adjacent grid cells.

Grid Creation

Step One: Setting up a Grid Layout

In the body of the HTML text file, create a div tag with a id titled corresponding to the layout of your grid. This will be the container for your grid. Inside the div, create a number of divs with different id titles for each grid item. Each of these will be the boxes inside the container.
Markup Example:

<body>
  <div id="layout">
    <div id="one">One</div>
    <div id="two">Two</div>
    <div id="three">Three</div>
    <div id="four">Four</div>
    <div id="five">Five</div>
  </div>
</body>

In the CSS style sheet, identify the container using the div id name and set the display to grid. The default grid display is vertically stacked grid items begining at the start of the cross axis.
Style Example: 

#layout{ display: grid;}

Step Two: Creating Grid Tracks

The next step is to create grid tracks. Grid tracks are the rows and columns of the grid. To create grid tracks, use the grid-template-columns and grid-template-rows properties. You can specify the size of each track using different units such as pixels, percentages, or fractions. It is important to understand the units of measurement in CSS, for additional help, check out CSS Tricky Concepts.
Style Example: 

#layout{
       display: grid;
       grid-template-rows: 100px 400px 100px;
       grid-template-columns: 200px 500px 200px;
       }
Developing Grid Areas

It may be difficult to track and read through numbers, which is why there is an option to name each of your rows and columns. It will also help with grid design planning as this example does:

#layout{
       display: grid;
       grid-template-rows: [header-start] 100px [content-start] 400px [footer-start] 100px;
       grid-template-columns: [ads] 200px [main] 500px [links] 200px;
       }

To assign the given names to the specific areas of the grid, use the grid-template-areas property. The allows developers to visually plan the grid structure.
Markup Example:

#layout{
       display: grid;
       grid-template-rows: [header-start] 100px [content-start] 400px [footer-start] 100px;
       grid-template-columns: [ads] 200px [main] 500px [links] 200px;
       grid-template-areas:
         "header header header"
         "ads    main   links"
         "footer footer footer";
       }
Short-hand Property

There is a shorthand property for the grid-template-rows, grid-template-columns, and grid-template-areas properties. The grid property allows you to set the rows, columns, and areas in one line. The rows and columns are seperated with a slash as shown in the example:

#layout{
       display: grid; 
       grid: 100px 400px 100px / 200px 225px 200px;

To add in the grid areas, use the order as follows: [start line name] "area names" track size. Repeat the order for each row in the grid then place a slash before begining the same process for the columns in the grid except without the areas names. Since they have been mentioned, it is not neccessary to restate the area names. Here is an example:

#layout{
       display: grid;
       grid: [header-start] "header header header" 100px
             [content-start] "ads main links" 400px
             [footer-start] "footer footer footer" 100px
            /[ads] 200px [main] 225px [links] 200px;
        }

Step Three: Placing Grid Items

To place grid items inside the grid layout, there are two methods that can be followed depending on preference:

Method One: Lines

This method places grid items using the grid lines. You can specify the starting and ending lines for each item using the grid-column and grid-row properties. In the stylesheet, use the id name given to each of the grid items to specify these four properties:

  • grid-row-start: The starting line for the grid item.
  • grid-row-end: The ending line for the grid item.
  • grid-column-start: The starting line for the grid item.
  • grid-column-end: The ending line for the grid item.

For example, if you want to place the grid item with the id of "one" in the first row and span the columns, you would use the following code:

#one{
    grid-row-start: 1;
    grid-row-end: 2;
    grid-column-start: 1;
    grid-column-end: 4;
   }
Method Two: Area

This method places grid items using the grid areas. You can specify the area for each item using the grid-area property. This is a shorthand property that combines the four properties above into one. For example, if you want to place the grid item with the id of "one" in the header area, you would use the following code:

#one{ grid-area: header; }

It can be used as a list of the grid lines for the item. This is seperated by slashes and appear in the following order: row-start, column-start, row-end, and 

#one{ grid-area: 1 / 1 / 2 / span 3; }

Grid Behavior

Grid items can be adjust to position, size, and align themselves within the grid. This can be done using the following properties:

Flow Direction

By default, grid items are placed in the order they appear in the HTML document. However, you can change the flow direction using the grid-auto-flow property. The possible values are:

  • row: The default value. Items are placed in rows.
  • column: Items are placed in columns.
  • dense: Items are placed in the first available space.

Spacing and Alignment

Grid items can be aligned using the same properties as flexbox. The following properties can be used to align grid items:

To align the grid items on the main axis, use the justify-content property which alters how space within the container moves around or between the grid items. This property can be used no matter what value is set to the grid-flow of the container. The alignment will apply to the container whether it is arranged horizontally or vertically. The property will be placed in the container id within the stylesheet. Here are just a few:

  • justify-content: start;
    • The start value positions the items at the start of the main axis.
  • justify-content: end;
    • The end value positions the items at the end of the main axis.
  • justify-content: center;
    • The center value positions the items in the center of the main axis.
  • justify-content: space-between;
    • The space-between value positions the items with space between them.
  • justify-content: space-around;
    • The space-around value positions the items with space around them.

To align all grid item on the cross axis, use the align-items property, which adjust the position and stretching of the items in the container. Here are just a few:

  • align-items: stretch;
    • The stretch value stretches the grid items from top to bottom depending on the container's set height value.
  • align-items: start;
    • The start value positions the grid items at the start of the cross axis.
  • align-items: end;
    • The end value positions the grid items at the end of the cross axis.
  • align-items: center;
    • The center value positions the grid items in the center of the cross axis.
  • align-items: left;
    • The left value positions the grid items to the left of the cross axis.
  • align-items: right;
    • The right value positions the grid items to the right of the cross axis.

In instances when there is leftover space within your grid container, use the align-content property, which modifies the alignment for entire grid trick. This property will be placed within the stylesheet in the grid's container id. Here are just a few:

  • align-content: start;
    • The start value positions the grid tracks at the start of the cross axis.
  • align-content: end;
    • The end value positions the grid tracks at the end of the cross axis.
  • align-content: center;
    • The center value positions the grid tracks in the center of the cross axis.
  • align-content: space-between;
    • The space-between value positions the grid tracks with space between them.
  • align-content: space-around;
    • The space-around value positions the grid tracks with space around them.

To align individual grid items on the cross axis, use the align-self property, which applies the set value to the specified items. The property is placed under a new class for the singular flex item in the stylesheet. Ensure that the class name matches the id given in the HTML markup. Here are just a few:

  • align-self: start;
    • The start value positions the grid item at the start of the cross axis.
  • align-self: end;
    • The end value positions the grid item at the end of the cross axis.
  • align-self: center;
    • The center value positions the grid item in the center of the cross axis.
  • align-self: left;
    • The left value positions the grid item to the left of the cross axis.
  • align-self: right;
    • The right value positions the grid item to the right of the cross axis.
